> An opinionated React Framework for building modern web applications.

## Quick Start

Install Dahlia CLI:

```bash
yarn global add dahlia-cli
```

New a Dahlia app:

```bash
dh new myapp
```

You will get:

```bash
.
├── README.md
├── package.json
├── pages
│   └── index.tsx
└── tsconfig.json
```

Run dev server:

```bash
cd myapp
dh start
```
