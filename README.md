## Quick Start

Install Dahlia CLI:

```bash
yarn global add dahlia-cli
```

New a Dahlia app:

```bash
dh new my-app
```

You will get:

```bash
.
├── README.md
├── node_modules
├── package.json
├── public
│   └── index.html
├── src
│   ├── pages
│   ├── routes.ts
│   └── index.tsx
└── tsconfig.json
```

Run dev server:

```bash
cd my-app
dh dev
```
